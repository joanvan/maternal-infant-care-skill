# 母婴全周期 Skill（OpenClaw）

面向母婴日常生活与健康管理的双语技能包，覆盖备孕/孕期/产后/新生儿/婴幼儿（0-3岁）。

[English](README.md)

## 覆盖内容

- 备孕与孕期分阶段支持
- 孕期营养与家常菜谱建议
- 妊娠糖尿病饮食周计划框架
- 孕期高血压风险观察要点
- 产后恢复与心理健康分级支持
- 母乳喂养深度指南（含接、堵奶、泵奶、储奶）
- 新生儿护理与发热分诊路径
- 婴幼儿喂养、辅食过敏引入、分月龄睡眠策略
- 0-36个月发育里程碑
- 家庭作息、母婴用品清单、每周家庭看板
- 30天计划模板（孕妇/婴幼儿）

## 安全声明

本技能用于健康教育、结构化计划和分诊支持，
**不能替代医生诊断或急诊救治**。
出现红旗症状请立即就医。

## 安装方式

### 方式A：安装到当前 workspace（推荐）

```bash
cp -r maternal-infant-care <你的workspace>/skills/
```

### 方式B：全局安装

```bash
cp -r maternal-infant-care ~/.openclaw/skills/
```

## 使用示例

- “我孕28周，帮我做30天饮食+作息+监测+复诊提醒计划。”
- “宝宝5个月夜醒多，请做30天喂养和睡眠计划。”
- “宝宝8个月鸡蛋过敏，做一份一个月辅食计划并给替代蛋白方案。”
