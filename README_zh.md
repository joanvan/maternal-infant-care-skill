# 母婴全周期 Skill（OpenClaw）

这是一个“可直接执行”的母婴技能包（中英双语），覆盖备孕/孕期/产后/新生儿/婴幼儿（0-3岁）。

[English](README.md)

## 特色

- 不只是纲要，包含实操SOP与步骤清单
- 先分诊再建议（红/黄/绿风险）
- 可落地的30天计划（按周+每日记录）
- 菜谱生成规则（支持控糖、补铁、过敏回避）
- 新生儿发热路径与就医摘要模板

## references 目录（双语）

- `references/zh/`：中文完整主题库（已细化）
- `references/en/`：英文核心实操模块

## 安装

```bash
cp -r maternal-infant-care <你的workspace>/skills/
```

## 使用示例

- “我孕28周，请做30天饮食+作息+监测+复诊提醒计划。”
- “宝宝2个月发热38.1°C，马上应该怎么处理？”
- “给我一个4周妊娠糖尿病菜谱，要求每天30分钟内能做完。”

## 安全声明

本技能用于健康教育与分诊支持，不能替代医生诊断。
出现红旗症状请立即就医。
