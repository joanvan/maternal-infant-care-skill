# Evidence & Guideline Map (English)

> This file anchors recommendation style and triage logic to major guidelines. It does not replace local clinical protocols.

## 1) Maternal care (preconception, pregnancy, postpartum)

### Primary sources
- WHO maternal and perinatal care guidance
- ACOG practice guidance (obstetric complications, prenatal/postpartum care)
- NICE guidance (hypertension in pregnancy, diabetes, intrapartum care)
- RCOG resources for high-risk obstetrics

### Skill modules using these
- `pregnancy.md`
- `hypertension-pregnancy-watch.md`
- `gestational-diabetes-meal-plan.md`
- `postpartum.md`

## 2) Newborn and infant care

### Primary sources
- AAP guidance (feeding, sleep safety, fever triage)
- CDC (immunization and developmental communication)
- WHO infant/young child feeding and growth monitoring
- NICE fever-in-children triage principles

### Skill modules using these
- `newborn-fever-pathway.md`
- `feeding-sleep.md`, `sleep-training-by-age.md`
- `development-vaccine.md`, `development-milestones-0-36m.md`
- `allergy-intro-food.md`

## 3) Nutrition and menu planning

### Primary sources
- WHO/FAO nutrition frameworks
- National dietary guidelines (region-specific)

### Skill modules using these
- `nutrition-pregnancy.md`, `recipes-pregnancy.md`
- `nutrition-infant.md`, `recipes-infant.md`
- `menu-generator-rules.md`

## 4) Mental health and family support

### Primary sources
- WHO mhGAP orientation
- Maternal mental health consensus guidance

### Skill modules using these
- `postpartum-mental-health.md`
- `daily-routines.md`, `weekly-family-dashboard.md`

## 5) Output discipline (must include)

1. Risk level (red/yellow/green)
2. Recheck timing
3. Escalation triggers
4. Non-replacement disclaimer (not a diagnosis)

## 6) Localization note

Vaccination schedules, care pathways, and medication access vary by country.
When user region is known, align recommendations to local standards.
