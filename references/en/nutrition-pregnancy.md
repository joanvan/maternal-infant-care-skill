# Pregnancy Nutrition (Operational)

## Daily framework
1. 3 meals + 2 snacks
2. Each meal: carbs + protein + vegetables
3. Hydration split across day (often ~1500-2000 ml/day, individualized)
4. Track appetite, symptoms, weekly weight trend

## Priority targets
- Protein in every main meal
- Iron-supportive meals + vitamin C pairing
- Calcium sources daily
- Fiber: vegetables + fruit daily

## Limit/avoid
- Raw/undercooked meat/eggs, unpasteurized dairy
- Excess sugar/salty ultra-processed foods
- Alcohol (avoid)

## Escalate if
- Persistent inability to eat/drink
- Dehydration signs
- Rapid worsening symptoms
