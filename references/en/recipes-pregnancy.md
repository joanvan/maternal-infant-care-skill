# Pregnancy Recipes (7-Day Practical Template)

## Use
- Pick target: balanced / iron support / glucose control
- Run breakfast-AM snack-lunch-PM snack-dinner format

## Day sample
- Breakfast: oatmeal + milk + egg + fruit
- AM snack: unsweetened yogurt
- Lunch: brown rice + steamed fish + broccoli
- PM snack: apple/pear
- Dinner: chicken + mixed vegetables + grain + light soup

## Rotation rule
- Do not repeat same main dish >2 times/week
- Keep prep <=30 min where possible

## Glucose-control swaps
- Refined rice -> mixed grains
- Sugary drinks -> water/unsweetened options
