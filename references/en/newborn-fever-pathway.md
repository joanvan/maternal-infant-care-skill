# Newborn/Young Infant Fever SOP

## Scope
- Infants 0-6 months (especially 0-3 months)

## Immediate check (within minutes)
- Temperature method/value
- Alertness
- Breathing effort (retractions/grunting/cyanosis)
- Feeding acceptance
- Urine output

## Age-based triage

### 0-3 months
- Fever >= 38.0 C -> urgent same-day ED evaluation

### 3-6 months
- If poor feeding, breathing issues, lethargy, low urine -> ED
- If clinically stable -> same-day clinic or close short-interval recheck

## Home observation only for low-risk cases
- Recheck temp every 2-4 hours
- Log feeds and diapers
- Escalate immediately if any deterioration
