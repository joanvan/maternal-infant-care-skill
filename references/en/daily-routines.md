# Maternal-Infant Daily Routine Templates

## Pregnancy weekday template
- Morning hydration, breakfast, short walk
- Midday balanced lunch + short rest
- Afternoon snack + hydration
- Evening light dinner + wind-down routine
- Consistent sleep window

## Newborn cycle template
- Feed -> burp -> soothe -> sleep
- Log feeds, diapers, temperature if needed

## Shift-work suggestion for caregivers
- Night shift A / Night shift B
- Day tasks split: feeding, cleaning, meal prep, resupply
