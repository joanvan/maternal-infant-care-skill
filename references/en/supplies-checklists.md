# Maternal-Infant Supplies Checklists

## Delivery bag
- IDs/records, maternal care items, baby clothing/diapers

## Home newborn kit
- Safe sleep setup, diapers/cleaning, thermometer, basic care tools

## Feeding kit
- Breastfeeding supplies and/or formula prep/sterilization tools

## Weekly restock
- Diapers/wipes, formula/food, essential meds and expiry checks
