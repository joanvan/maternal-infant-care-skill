# 30-Day Plan Template (Detailed)

## A) Pregnancy monthly plan

### Inputs
- Gestational week
- Main goal: glucose/BP/anemia/sleep/anxiety
- Constraints: budget, cooking time, food restrictions

### Weekly block (Week 1-4)
1. Weekly goals
2. Daily non-negotiables (water, sleep, movement, supplements)
3. 7-day menu
4. Monitoring table (BP/weight/glucose/movement)
5. Recheck milestones
6. Escalation triggers

### Daily log
- Date
- Sleep hours
- Water intake
- Activity
- Meals adherence
- Monitoring values
- Notes

## B) Infant monthly plan

### Inputs
- Age in months
- Feeding mode
- Main goal: sleep/feeding/allergy/growth/development

### Weekly block (Week 1-4)
1. Feeding target
2. Sleep target
3. Development play tasks
4. New-food schedule (if applicable)
5. Observation indicators (temp/urine/stool/alertness)
6. Vaccine/well-visit reminders
