# Pregnancy 2-Week Menu Cases (by trimester)

> Daily structure: breakfast - snack - lunch - snack - dinner. Swap equivalent foods by preference.

## A) First trimester (2 weeks)

### Week 1
- D1: Oatmeal + milk + egg / yogurt / chicken noodle soup + greens / banana / tofu-shrimp rice bowl
- D2: Whole-grain toast + egg / small nuts / tomato-beef rice / apple / steamed fish + broccoli
- D3: Millet porridge + egg custard / yogurt / mushroom-chicken noodles / pear / pumpkin minced beef rice
- D4-D7: rotate D1-D3; avoid repeating same main dish >2 times in a row

### Week 2
- D8-D14: keep Week 1 structure, rotate protein sources (fish/chicken/tofu/lean beef), maintain nausea-friendly meals

## B) Second trimester (2 weeks)

### Week 1
- D1: Oatmeal + milk + egg / orange / brown rice + steamed fish + broccoli / yogurt / chicken quinoa bowl
- D2: Whole-grain sandwich + milk / nuts / tomato beef stew + rice / apple / tofu veggie noodles
- D3: Corn porridge + egg / kiwi / mushroom chicken rice / yogurt / salmon + pumpkin + greens
- D4-D7: rotate D1-D3 pattern

### Week 2
- D8-D14: increase iron-supportive meals per clinician advice; include leafy greens daily

## C) Third trimester (2 weeks)

### Week 1
- D1: Oatmeal + egg + fruit / yogurt / steamed fish + mixed grains + greens / pear / tofu lean-meat soup + millet rice
- D2: Whole-grain bread + milk / small nuts / chicken veggie rice / banana / shrimp egg + vegetables
- D3: Sweet potato porridge + egg custard / yogurt / beef veggie noodles / apple / cod + pumpkin mash + spinach
- D4-D7: rotate; keep dinners lighter

### Week 2
- D8-D14: reduce high-sodium/greasy dinners; use light bedtime snack if needed (e.g., warm milk + whole-grain cracker)

## Execution rules
- Keep hydration split across the day (often ~1500-2000 ml/day, individualized)
- Include protein in every main meal
- Switch to dedicated plans if glucose/BP concerns arise
