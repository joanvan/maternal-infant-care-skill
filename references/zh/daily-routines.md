# 母婴日常作息（执行模板）

## 1) 孕妇工作日模板

- 07:00 起床补水
- 08:00 早餐（蛋白+主食+蔬果）
- 10:30 加餐 + 10-20分钟步行
- 12:30 午餐
- 13:30-14:00 短休
- 15:30 加餐 + 补水
- 18:30 晚餐
- 20:00 放松（拉伸/泡脚/阅读）
- 22:30 睡眠

## 2) 新生儿家庭模板（按循环）

- 喂养 -> 拍嗝 -> 安抚 -> 睡眠
- 每循环记录：喂养量、尿布、体温（必要时）

## 3) 轮班分工建议

- 夜班A：22:00-03:00
- 夜班B：03:00-07:00
- 白天：按任务板轮转（喂养/清洁/备餐/补货）

## 4) 每日最少记录项

- 妈妈：睡眠小时、情绪评分、体温/症状
- 宝宝：喂养次数、尿布次数、睡眠总时长

## 5) 触发调整规则

- 连续3天睡眠严重不足 -> 调整轮班
- 连续2天记录异常（喂养/尿量/体温） -> 联系医生
