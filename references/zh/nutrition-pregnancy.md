# 孕期营养（阈值+步骤版）

> 以下为通用教育范围，个体化方案以产科/营养门诊为准。

## 1) 每日执行框架（先照做）

1. 三餐 + 2次加餐（避免长时间空腹）
2. 每餐都要有：主食 + 蛋白 + 蔬菜
3. 饮水分次完成（通常 1500-2000ml/天，按医嘱个体化）
4. 每日记录：食欲、体重（每周）、不适症状

## 2) 关键营养目标（实操）

- 蛋白：每餐有明确蛋白来源（蛋/奶/鱼/禽/豆）
- 铁：每周安排红肉/动物肝（按医嘱）+ 维C食物同餐
- 钙：奶制品或替代来源每天稳定摄入
- 纤维：每日至少2种蔬菜 + 1-2份水果

## 3) 需要限制的项目

- 生食/半生食、未熟蛋肉、未巴氏消毒奶制品
- 高糖饮料、过咸加工食品
- 咖啡因过量（按医生建议）
- 酒精（建议避免）

## 4) 特殊场景处理

### 妊娠糖尿病
- 主食分配到三餐与加餐，减少单次碳水峰值
- 餐后10-20分钟轻活动（医生允许）
- 参考：`gestational-diabetes-meal-plan.md`

### 便秘
- 饮水 + 纤维 + 轻活动同步执行3-7天再评估

### 严重孕吐
- 先补液与电解质，再逐步恢复固体食物
- 持续无法进食/脱水征象需就医

## 5) 升级就医阈值

- 连续多天几乎不能进食
- 明显体重快速下降或脱水征象
- 合并头晕、心悸、尿量显著减少
