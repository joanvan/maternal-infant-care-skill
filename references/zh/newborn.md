# 新生儿护理（0-28天，实操版）

## 1) 每日观察四件事（固定记录）

1. 喂养：次数、每次时长/奶量、是否吞咽顺畅  
2. 排泄：湿尿布次数、大便颜色和性状  
3. 体温：每天固定1-2次，异常时加测  
4. 状态：精神反应、哭声、肤色、呼吸是否平稳

## 2) 喂养与排泄参考

- 湿尿布：生后第5天后通常≥6片/天
- 大便：母乳宝宝次数可多可少，关键看精神和体重趋势
- 体重：生后10-14天多数可回到出生体重

## 3) 黄疸家庭观察（可执行）

### 观察频次
- 每天至少2次（自然光下）观察皮肤/眼白黄染范围

### 升级规则
- 黄染范围从面部向四肢扩展
- 精神差、嗜睡、吃奶变差
- 合并发热或哭声异常

出现以上任一条：尽快就医评估黄疸水平。

## 4) 脐部护理 SOP

1. 保持清洁干燥
2. 洗澡后轻柔擦干脐周
3. 尿布边缘避开脐部
4. 不自行涂抹刺激性药物

### 就医触发
- 脐周红肿扩散
- 渗液有异味
- 触碰明显疼痛/伴发热

## 5) 安全睡眠 SOP

- 始终仰卧睡
- 独立睡眠空间（同室不同床）
- 床内避免枕头、毛绒玩具、松软被褥
- 室温适中，避免过热包裹

## 6) 立即急诊触发（任一项）

- 体温异常伴反应差
- 呼吸急促、胸凹、口唇发紫
- 持续拒奶、喷射样呕吐、尿量明显减少
- 惊厥或叫不醒
