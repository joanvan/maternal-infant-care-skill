# 孕期两周菜谱案例（分孕期）

> 每日结构：早餐-加餐-午餐-加餐-晚餐。可按口味替换同类食材。

## A. 孕早期两周（示例）

### 第1周
- D1: 燕麦牛奶鸡蛋 / 酸奶 / 鸡丝面+青菜 / 香蕉 / 豆腐虾仁煲+杂粮饭
- D2: 全麦吐司+煎蛋 / 坚果少量 / 番茄牛肉饭 / 苹果 / 清蒸鱼+西兰花
- D3: 小米粥+鸡蛋羹 / 酸奶 / 香菇鸡肉面 / 梨 / 南瓜牛肉末饭
- D4-D7: 轮换D1-D3，主菜不连续重复>2次

### 第2周
- D8-D14: 在第1周基础上替换蛋白来源（鱼/鸡/豆腐/瘦牛）并维持清淡易消化

## B. 孕中期两周（示例）

### 第1周
- D1: 燕麦+牛奶+水煮蛋 / 橙子 / 糙米饭+清蒸鱼+西兰花 / 酸奶 / 鸡胸藜麦沙拉
- D2: 全麦三明治+奶 / 坚果 / 番茄牛肉炖土豆+米饭 / 苹果 / 豆腐青菜面
- D3: 玉米粥+鸡蛋 / 猕猴桃 / 香菇鸡腿饭 / 酸奶 / 三文鱼+南瓜+时蔬
- D4-D7: 按D1-D3轮换

### 第2周
- D8-D14: 增加含铁主菜频次（按医嘱），每天至少一餐深绿叶菜

## C. 孕晚期两周（示例）

### 第1周
- D1: 燕麦+蛋+水果 / 酸奶 / 清蒸鱼+杂粮饭+青菜 / 梨 / 豆腐瘦肉汤+小米饭
- D2: 全麦面包+奶 / 坚果少量 / 鸡胸肉蔬菜饭 / 香蕉 / 虾仁滑蛋+蔬菜
- D3: 红薯粥+鸡蛋羹 / 酸奶 / 牛肉蔬菜面 / 苹果 / 鳕鱼+南瓜泥+菠菜
- D4-D7: 轮换并保持晚餐清淡

### 第2周
- D8-D14: 晚餐减少高盐和油炸，睡前加餐以温奶/全麦饼干为主

## 执行规则
- 每天饮水分次完成（常见1500-2000ml，遵医嘱）
- 每餐都要有蛋白质
- 出现血糖/血压异常，切换专项菜单
