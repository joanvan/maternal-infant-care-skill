# 孕妇菜谱（可执行7天模板）

## 使用方法

- 先选目标：均衡 / 补铁 / 控糖 / 改善便秘
- 每天按“早-加-午-加-晚”执行
- 每天记录：饱腹感、餐后不适、是否可持续

## 7天模板（30分钟内优先）

### Day1
- 早：燕麦牛奶+鸡蛋+香蕉
- 加：无糖酸奶
- 午：糙米饭+清蒸鱼+西兰花
- 加：苹果
- 晚：鸡胸肉炒彩椒+杂粮饭+紫菜汤

### Day2
- 早：全麦吐司+煎蛋+牛奶
- 加：坚果少量
- 午：番茄牛肉炖土豆+青菜
- 加：梨
- 晚：豆腐虾仁煲+小米饭

### Day3
- 早：小米粥+鸡蛋羹+猕猴桃
- 加：奶酪/酸奶
- 午：藜麦饭+香菇鸡肉+时蔬
- 加：橙子
- 晚：鳕鱼+南瓜+清炒菠菜

### Day4-7
- 按 Day1-3 轮换，主菜不连续重复>2次

## 控糖版替换规则

- 白米饭 -> 糙米/燕麦/杂豆饭
- 甜饮 -> 白水/无糖饮品
- 每餐主食减量，蛋白和蔬菜比例提高

## 补铁版加成

- 每周2-3次瘦红肉（按医嘱）
- 同餐加富含维C食材
