# 孕妇家常菜谱（示例）

> 目标：高蛋白、易消化、低油盐；口味按个人调整。

## 早餐方案（任选）

1. 鸡蛋蔬菜三明治 + 牛奶 + 猕猴桃
2. 小米南瓜粥 + 清蒸鸡蛋羹 + 坚果
3. 燕麦酸奶杯（燕麦+酸奶+蓝莓+香蕉）

## 午晚餐方案（任选）

1. 番茄牛肉炖土豆 + 糙米饭 + 西兰花
2. 清蒸鳕鱼 + 山药胡萝卜 + 紫米饭
3. 香菇鸡肉炖豆腐 + 菠菜 + 杂粮饭
4. 虾仁滑蛋 + 清炒芦笋 + 玉米饭

## 汤类

- 冬瓜虾皮豆腐汤
- 番茄菌菇鸡汤
- 海带排骨汤（控盐）

## 加餐

- 无糖酸奶 + 水果
- 全麦饼干 + 温奶
- 一小把坚果（避免过量）

## 简易一周模板

- 周一三五：鱼类+豆制品
- 周二四六：禽肉/瘦牛肉
- 周日：杂豆+蛋奶为主
- 每天至少2种蔬菜+1-2份水果
